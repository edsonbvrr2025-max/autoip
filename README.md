# autoip V 2.4
Troca automática de IP usando o tor

Como instalar essa ferramenta e suas dependências:

* Requisitos:
  sudo apt install -y python3
  sudo apt-get install tor

1: git clone https://github.com/edsonbvrr2025-max/autoip.git

2 : cd autoip

✅ Faça um comando por vez (passo a passo)

3 : Instalar suporte a venv (uma vez só)

apt install python3-venv -y

4 : Criar o ambiente virtual

python3 -m venv venv

5 : Ativar o ambiente

source venv/bin/activate

👉 Vai aparecer assim:

(venv) root㉿

6 : Instalar requests

pip install requests

7 : Agora sim rode o script:

python3 install.py

[+] para instalar pressione (Y) para desinstalar pressione (N)

Aperte y para instalar

8 : Agora inicie a ferramnta com o script:

python3 autoip.py

9 : ⏱️  Intervalo (segundos):

3 segundos
